# Changelog

## Version 3.0 - Serverless Visitor Counter

### Added
- Added serverless visitor counter to the portfolio website.
- Created DynamoDB table to store visitor count.
- Created Lambda function to update and return visitor count.
- Created API Gateway HTTP API endpoint for frontend integration.
- Integrated visitor count display in the hero section.

### Updated
- Connected frontend JavaScript with API Gateway endpoint.
- Deployed latest changes using GitHub Actions CI/CD.
